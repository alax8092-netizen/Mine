A premium, luxury, responsive website built for my girlfriend.

## How to use:
1. Place your `photo1.jpg` in the root folder.
2. Place your `music.mp3` in the root folder.
3. Push everything to GitHub Pages.

- Glassmorphism design
- Live Anniversary Timer
